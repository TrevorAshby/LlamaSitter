# Code of Conduct

## Our Pledge

We want LlamaSitter to be a welcoming, respectful project for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Focus feedback on the work, not the person.
- Help keep the project accessible to new contributors.

## Unacceptable Behavior

- Harassment or discrimination
- Personal attacks
- Bad-faith disruption
- Publishing private information without permission

## Enforcement

Project maintainers may remove or reject content that violates this code of conduct.

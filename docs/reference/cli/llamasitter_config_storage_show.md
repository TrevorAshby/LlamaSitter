# llamasitter config storage show

Show storage settings

## Examples

```text
llamasitter config storage show
  llamasitter config storage show --output yaml
```

## Usage

```text
Usage:
  llamasitter config storage show [flags]

Examples:
  llamasitter config storage show
  llamasitter config storage show --output yaml

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config ui enable

Enable the dashboard UI listener

## Examples

```text
llamasitter config ui enable
  llamasitter config ui enable --dry-run
```

## Usage

```text
Usage:
  llamasitter config ui enable [flags]

Examples:
  llamasitter config ui enable
  llamasitter config ui enable --dry-run

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

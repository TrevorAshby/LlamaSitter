# llamasitter desktop logs

Inspect the app-managed desktop logs path

## Usage

```text
Usage:
  llamasitter desktop logs [flags]
  llamasitter desktop logs [command]

Available Commands:
  path        Print the app-managed logs directory

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")

Use "llamasitter desktop logs [command] --help" for more information about a command.
```

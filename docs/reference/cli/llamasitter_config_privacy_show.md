# llamasitter config privacy show

Show privacy and redaction settings

## Examples

```text
llamasitter config privacy show
  llamasitter config privacy show --output json
```

## Usage

```text
Usage:
  llamasitter config privacy show [flags]

Examples:
  llamasitter config privacy show
  llamasitter config privacy show --output json

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config path

Print the resolved path of the target config file

## Examples

```text
llamasitter config path
  llamasitter config path --config /tmp/llamasitter.yaml
```

## Usage

```text
Usage:
  llamasitter config path [flags]

Examples:
  llamasitter config path
  llamasitter config path --config /tmp/llamasitter.yaml

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

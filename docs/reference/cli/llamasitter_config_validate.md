# llamasitter config validate

Validate the selected config file without contacting upstreams

## Examples

```text
llamasitter config validate
  llamasitter config validate --output json
```

## Usage

```text
Usage:
  llamasitter config validate [flags]

Examples:
  llamasitter config validate
  llamasitter config validate --output json

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter desktop config

Inspect the app-managed desktop config path

## Usage

```text
Usage:
  llamasitter desktop config [flags]
  llamasitter desktop config [command]

Available Commands:
  path        Print the app-managed config path

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")

Use "llamasitter desktop config [command] --help" for more information about a command.
```

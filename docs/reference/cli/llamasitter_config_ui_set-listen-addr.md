# llamasitter config ui set-listen-addr

Set the UI listener bind address

## Examples

```text
llamasitter config ui set-listen-addr 127.0.0.1:11439
```

## Usage

```text
Usage:
  llamasitter config ui set-listen-addr HOST:PORT [flags]

Examples:
  llamasitter config ui set-listen-addr 127.0.0.1:11439

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

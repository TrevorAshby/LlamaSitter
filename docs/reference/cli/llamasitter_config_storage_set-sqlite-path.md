# llamasitter config storage set-sqlite-path

Set the SQLite database path

## Examples

```text
llamasitter config storage set-sqlite-path ~/.llamasitter/custom.db
```

## Usage

```text
Usage:
  llamasitter config storage set-sqlite-path PATH [flags]

Examples:
  llamasitter config storage set-sqlite-path ~/.llamasitter/custom.db

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config privacy add-redact-header

Add one header name to redact_headers

## Examples

```text
llamasitter config privacy add-redact-header authorization
```

## Usage

```text
Usage:
  llamasitter config privacy add-redact-header NAME [flags]

Examples:
  llamasitter config privacy add-redact-header authorization

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

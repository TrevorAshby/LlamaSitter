# llamasitter completion

Generate shell completion scripts

## Examples

```text
llamasitter completion zsh > ~/.zsh/completions/_llamasitter
  llamasitter completion bash > /usr/local/etc/bash_completion.d/llamasitter
```

## Usage

```text
Usage:
  llamasitter completion [bash|zsh|fish|powershell] [flags]

Examples:
  llamasitter completion zsh > ~/.zsh/completions/_llamasitter
  llamasitter completion bash > /usr/local/etc/bash_completion.d/llamasitter

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config storage

Inspect and update storage settings

## Usage

```text
Usage:
  llamasitter config storage [flags]
  llamasitter config storage [command]

Available Commands:
  set-sqlite-path Set the SQLite database path
  show            Show storage settings

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")

Use "llamasitter config storage [command] --help" for more information about a command.
```

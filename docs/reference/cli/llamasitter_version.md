# llamasitter version

Print LlamaSitter build version metadata

## Examples

```text
llamasitter version
  llamasitter version --output json
```

## Usage

```text
Usage:
  llamasitter version [flags]

Examples:
  llamasitter version
  llamasitter version --output json

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

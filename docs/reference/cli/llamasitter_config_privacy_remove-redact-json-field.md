# llamasitter config privacy remove-redact-json-field

Remove one field name from redact_json_fields

## Examples

```text
llamasitter config privacy remove-redact-json-field messages
```

## Usage

```text
Usage:
  llamasitter config privacy remove-redact-json-field NAME [flags]

Examples:
  llamasitter config privacy remove-redact-json-field messages

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter desktop db

Inspect the app-managed desktop database path

## Usage

```text
Usage:
  llamasitter desktop db [flags]
  llamasitter desktop db [command]

Available Commands:
  path        Print the app-managed SQLite database path

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")

Use "llamasitter desktop db [command] --help" for more information about a command.
```

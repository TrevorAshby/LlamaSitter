# llamasitter config view

Render the current config as yaml, json, or a summary table

## Examples

```text
llamasitter config view
  llamasitter config view --output json
```

## Usage

```text
Usage:
  llamasitter config view [flags]

Examples:
  llamasitter config view
  llamasitter config view --output json

Flags:
      --output string   output format: table, json, or yaml (default "yaml")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter desktop

Inspect macOS desktop app-managed paths

## Usage

```text
Usage:
  llamasitter desktop [flags]
  llamasitter desktop [command]

Available Commands:
  config      Inspect the app-managed desktop config path
  db          Inspect the app-managed desktop database path
  logs        Inspect the app-managed desktop logs path

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")

Use "llamasitter desktop [command] --help" for more information about a command.
```

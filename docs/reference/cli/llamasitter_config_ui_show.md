# llamasitter config ui show

Show UI enablement and listen address

## Examples

```text
llamasitter config ui show
  llamasitter config ui show --output json
```

## Usage

```text
Usage:
  llamasitter config ui show [flags]

Examples:
  llamasitter config ui show
  llamasitter config ui show --output json

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

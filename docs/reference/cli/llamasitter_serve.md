# llamasitter serve

Start the proxy, storage, API, and dashboard services

## Examples

```text
llamasitter serve --config llamasitter.yaml
  llamasitter serve --config /Users/me/Library/Application Support/LlamaSitter/llamasitter.yaml
```

## Usage

```text
Usage:
  llamasitter serve [flags]

Examples:
  llamasitter serve --config llamasitter.yaml
  llamasitter serve --config /Users/me/Library/Application Support/LlamaSitter/llamasitter.yaml

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config listener list

List configured listeners

## Examples

```text
llamasitter config listener list
  llamasitter config listener list --output json
```

## Usage

```text
Usage:
  llamasitter config listener list [flags]

Examples:
  llamasitter config listener list
  llamasitter config listener list --output json

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

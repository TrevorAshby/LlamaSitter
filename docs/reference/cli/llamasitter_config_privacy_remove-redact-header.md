# llamasitter config privacy remove-redact-header

Remove one header name from redact_headers

## Examples

```text
llamasitter config privacy remove-redact-header authorization
```

## Usage

```text
Usage:
  llamasitter config privacy remove-redact-header NAME [flags]

Examples:
  llamasitter config privacy remove-redact-header authorization

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config ui disable

Disable the dashboard UI listener

## Examples

```text
llamasitter config ui disable
  llamasitter config ui disable --dry-run
```

## Usage

```text
Usage:
  llamasitter config ui disable [flags]

Examples:
  llamasitter config ui disable
  llamasitter config ui disable --dry-run

Flags:
      --dry-run   print the updated config without writing it

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```

# llamasitter config ui

Inspect and update dashboard UI settings

## Usage

```text
Usage:
  llamasitter config ui [flags]
  llamasitter config ui [command]

Available Commands:
  disable         Disable the dashboard UI listener
  enable          Enable the dashboard UI listener
  set-listen-addr Set the UI listener bind address
  show            Show UI enablement and listen address

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")

Use "llamasitter config ui [command] --help" for more information about a command.
```

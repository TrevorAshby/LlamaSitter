# llamasitter config listener show

Show one configured listener

## Examples

```text
llamasitter config listener show default
  llamasitter config listener show default --output yaml
```

## Usage

```text
Usage:
  llamasitter config listener show NAME [flags]

Examples:
  llamasitter config listener show default
  llamasitter config listener show default --output yaml

Flags:
      --output string   output format: table, json, or yaml (default "table")

Global Flags:
  -c, --config string   path to config file (default "llamasitter.yaml")
```
